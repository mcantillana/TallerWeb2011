Descripción del curso:

Para todos es cierto que Internet es un medio de comunicación económico y de alcance mundial. Sin embargo, no todos cuentan con el conocimiento para desarrollar un buen sitio web. Conceptos como hosting, registros de dominio y otros son desconocidos para la mayoría, incluso para algunos "iniciados".
Por este motivo, es de gran importancia que todo profesional tenga una noción de lo que es un sitio web y del provecho que se le puede sacar a este conocimiento, en cualquier ámbito del conocimiento.

Objetivo general del curso:

Desarrollar un proyecto web.

Objetivos específicos del curso:

  • Comprender los conceptos relacionados con el desarrollo de sitios web.
  • Conocer y aplicar los conceptos de usabilidad y accesibilidad de sitios web.
  • Utilizar herramientas de gráfica y gestión de páginas HTML.
  • Utilizar herramientas específicas de la gestión de archivos a distancia.

Contenidos del curso

  • Conceptualización de un desarrollo web
  • Conceptos de usabilidad y accesibilidad
  • Necesidades del usuario
  • Diseño de interacción y arquitectura de información
  • Diseño visual de un sitio
  • Registro y costos asociados al desarrollo

Edición y publicación de páginas web

  • Lenguaje HTML
  • Diseño de páginas
    • Banners y botones
    • Manejo de imágenes
    • Manejo de textos
  • Links

Creación de sitios

  • Organización de sitios (Arquitectura de Información)
  • Estilo visual de un sitio, utilizando CSS
